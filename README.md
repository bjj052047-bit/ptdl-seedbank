# PTDL 연구실 포털

세종대학교 스마트생명산업융합학과 PTDL 연구실의 통합 운영 웹사이트입니다. 종자실 재고 관리와 실험실 예약을 하나의 로그인으로 이용합니다.

## 기술 스택
- **프론트엔드**: Next.js (React)
- **데이터베이스 + 로그인**: Supabase (Postgres + Auth + Row Level Security)
- **배포**: Vercel

## 진행 상황
- [x] 데이터베이스 스키마 설계 (`supabase/schema.sql`)
- [x] 로그인 / 회원가입 (이메일 + 비밀번호) + 관리자 승인형 가입
- [x] 역할 기반 접근 제한 (담당자 / 승인자 / 연구원, 서버가 강제하는 진짜 권한)
- [x] 종자실 - 검색 / 조회 탭
- [x] 종자실 - 종자 요청 탭
- [x] 종자실 - 입출고 기록 탭 (담당자 전용)
- [x] 종자실 - 데이터 등록·관리 탭 (담당자 전용)
- [x] 실험실 예약 (405B / 311C, 월간 캘린더, 1시간 단위 예약)
- [x] 가입 승인 관리 탭 (담당자 / 승인자 전용)
- [x] 배드 예약 (Growth Room 등 8개 시설 42개 배드, 기간 단위 예약, 겹침 허용 + 확인 알림)

## 가입 승인 흐름
1. 새 계정이 이메일 + 비밀번호 + 이름으로 가입하면 `profiles.status`가 `pending`으로 시작합니다.
2. 승인 전에는 어떤 탭도 사용할 수 없고 "승인 대기중" 화면만 보입니다.
3. 담당자(staff) 또는 승인자(supervisor)가 **가입 승인** 탭에서 승인/거절을 처리합니다.
4. 승인되면 새로고침 후 바로 모든 탭을 이용할 수 있습니다.

## 실험실 예약 규칙
- 실험실은 405B, 311C 두 곳입니다.
- 하루 24시간, 1시간 단위로 예약합니다.
- 같은 실험실·같은 시간대는 한 팀만 예약할 수 있습니다 (겹치는 예약은 서버가 자동으로 막습니다).
- 본인 예약은 언제든 취소할 수 있고, 담당자/승인자는 다른 사람의 예약도 취소할 수 있습니다.

## 배드 예약 규칙
- 배드는 8개 시설(Growth Room 415, Incubator 415, Growth Chamber 415, Training Farm Greenhouse/Outdoor, Rapid Generation Advancement Room, Dark Room, Hydroponics Growth Room 454)에 걸쳐 총 42개입니다.
- 실험실과 달리 시작일~종료일 기간 단위로 예약하고, 한 배드를 여러 팀이 나눠 쓸 수 있어 겹치는 예약도 허용됩니다.
- 다만 겹치는 예약을 등록하려 하면 "OO 기간과 겹칩니다. 공유하도록 협의하셨나요?" 확인창이 떠서, 협의 여부를 한 번 더 확인시킵니다.
- 본인 예약은 언제든 취소할 수 있고, 담당자/승인자는 다른 사람의 예약도 취소할 수 있습니다.
